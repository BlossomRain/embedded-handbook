# Project Overview

## Goal

Maintain the embedded handbook as an independent knowledge project for device notes, component principles, and engineering reference material.

## Scope

- in scope: device notes, technology introductions, diagrams, writing standards, templates, and handbook publishing material
- out of scope: treating this repository as a child of `stm32-workspace`

## Current Phase

- phase: independent knowledge project governance
- next milestone: keep handbook content and documentation standards aligned under the new repository root

## Key Assets

- main source root: `content/`
- main docs root: `docs/`
- main templates root: `templates/`
- main support scripts/tools root: `tools/`
